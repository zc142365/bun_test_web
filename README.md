# bun_test_web
bun과 express.js를 활용한 test web

## 소개
bun과 express.js를 활용한 학습용 test web 입니다

## 개발 환경
  - OS : WIndows
  - Development Language : javascript
  - Library : express.js
  - Tools : VSCode
  - Engine : Bun
## 기능
단순한 테스트용 웹

## 사용방법
bun 공식 : https://bun.sh/docs/installation
에서 bun 설치 후
bun add
bun index.ts
사용
